HTML 5 - Ajuda para lembrar das TAG's

Veremos aqui como são as tags para uma tabela usando o HTML 5

Table / Tabela
TAG Descrição Tradução
<table> Tag principal de uma tabela table = tabela
<tr> Define uma linha table row = linha da tabela
<td> Informação da celula table data = dado da tabela



Text / Texto
TAG Descrição Tradução
<p> Define o espaço para um paragrafo paragraf = paragrafo
<h1>, <h2>, <h3>,... <h6> Define titulos, vai de 1 até 6
sendo o numero 1 o mais chamativo		 h1 = heading 1 = cabeçalho 1
<br> pula uma linha br = break line = quebrar linha
<strong> Torna o texto mais espeço strong = forte / reforçar
<em> Enfatiza o texto deixando em italico em = emphasize = enfatizar
